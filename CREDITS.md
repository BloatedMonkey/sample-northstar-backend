- **Clean Code**: SOLID principles, design patterns, maintainability

## Technology Stack

This project leverages the following open-source technologies:

- **NestJS** - Progressive Node.js framework
- **TypeScript** - Typed JavaScript
- **PostgreSQL** - Advanced relational database
- **Prisma** - Next-generation ORM
- **Redis** - In-memory data structure store
- **BullMQ** - Background job processing
- **Passport.js** - Authentication middleware
- **Pino** - Fast logging library

All dependencies are properly licensed and credited to their respective authors.
